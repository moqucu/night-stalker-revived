Screen Resolution & Sprite Sizes
--------------------------------

-   Screen resolution

    -   Original: 160 x 196

    -   Translated: 4 \*160 = 640 x 392 = 2\*196

-   Sprite sizes

    -   Original:

        -   Small: 8 x 8

        -   Large: 8 x 16

    -   Translated:

        -   Small: 32 x 16

        -   Large: 32 x 32
